
Project: Online Library

My Online Library is a virtual bookstore web application that allows users to browse, borrow, and return books online. It provides an interactive and user-friendly interface where users can easily explore available titles and authors of the books they need to borrow. The system keeps track of book availability and user activity to ensure a smooth borrowing experience.

Tech used:

HTML, CSS, JavaScript, Node.js, express.js, and MongoDB

Lessons Learned:

While building this project, I learned how to integrate front-end and back-end technologies to create a complete and functional web application.
